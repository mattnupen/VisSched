# VisSched

A free, account-free **visual schedule and class timer** for K–12 classrooms, special education, therapy sessions, and home routines.

Build an ordered list of timed activity blocks ("Warm-up 5m / Mini-lesson 10m / Work time 20m / Recap 10m"), then project it on a screen. Students see what's happening *now*, what's *next*, and how long is *left* — all in one view.

## Try it

**→ [vissched.com](https://vissched.com)**

No signup, no accounts, no tracking. It runs entirely in your browser.

## Features

- **12 view modes** — schedule list, growing sunflower, rocket launch, melting candle, train to station, and more.
- **5 color themes** and **6 text sizes** sized to be legible from the back row.
- **Optional transition chimes** at 2:00, 1:00, 0:30, and 0:10 remaining.
- **Share by URL** — send a schedule to a colleague with one link. No accounts needed.
- **Offline-capable** — once loaded, the tab can run all day without network.
